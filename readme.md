## 📌 Project Overview

> This project was completed as part of my internship at **Data Zenix**, where I performed end-to-end **Exploratory Data Analysis (EDA)** on real-world retail and warehouse sales data. The goal was to uncover patterns, trends, and actionable insights from raw transactional data.

---

## 📂 Repository Structure

```
📦 retail-sales-eda/
 ┣ 📓 retail_eda.ipynb          → Full EDA notebook (step-by-step analysis)
 ┣ 📊 cleaned_retail_data.csv   → Cleaned & merged dataset
 ┣ 🖼️ monthly_trends.png        → Monthly sales trend visualization
 ┗ 🖼️ top_products.png          → Top 5 product category bar chart
```
